# Modelo de Negócio Simulado

Este projeto modela decisões financeiras reais enfrentadas por empresas.

## Conceitos Modelados

- Capital de giro
- Prazo médio de recebimento
- Burn rate diário
- Risco de quebra por desalinhamento de caixa

## Problema Central

Lucro contábil não garante sobrevivência operacional.

O sistema avalia se a empresa consegue se manter viva até que a receita seja efetivamente recebida.

## Impacto da Escala

Aumento de investimento gera pressão imediata no caixa.
O retorno ocorre com atraso.

O simulador responde se a escala é financeiramente segura.
