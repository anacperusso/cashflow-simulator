# Exemplos de Uso

## Cenário Base
Simulação com parâmetros atuais para avaliar sustentabilidade.

## Cenário de Escala
Aumento proporcional de receita e custo para avaliar impacto no caixa.

## Cenário de Quebra
Identificação do dia exato em que o caixa se torna negativo.

Cada cenário retorna:
- Dias até quebra (se houver)
- Caixa final
- Classificação de risco
